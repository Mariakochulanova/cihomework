# cihomework
Счётчик INSTRUCTIONS отвечает за предоставление информации о том, какой объём кода был выполнен или наоборот - не выполнен. Данная метрика не зависит от форматирования кода.
Счётчик BRANCH предоставляет информацию о покрытии веток с операторами if и switch, то есть показывает общее количество таких ветвей в коде и насколько каждая из них выполнена 
или пропущена. 
Счётчик LINE показывает данные о покрытии строк кода, для файлов, скомпилированных с отладочной информацией.
